# Private Signal Board Bounty Handoff

This workspace is a Midnight #314-oriented package built from the official bulletin-board dApp scaffold.
It adds a small off-chain metadata service and a React metadata panel so the tutorial can explain the
split between Compact public ledger state, private witness state, and non-sensitive backend metadata.

## What Changed

- Added `metadata-api/`, a small Node HTTP service with `/health`, `/campaign`, and per-contract metadata routes.
- Added `bboard-ui/src/components/MetadataPanel.tsx` and wired it into the first screen.
- Added `VITE_METADATA_API_URL` to the UI env files.
- Upgraded root `@midnight-ntwrk/compact-runtime` to `^0.16.0` to match code generated by Compact compiler `0.31.0`.
- Added metadata API coverage to `.github/workflows/ci.yaml`.
- Documented metadata API build/start steps in `README.md`.
- Pushed the public repository to `https://github.com/marmar9615-cloud/midnight-private-signal`.
- Saved an unpublished DEV draft at `https://dev.to/marmar9615cloud/building-private-signal-board-a-full-stack-midnight-dapp-with-compact-witnesses-react-and-32ef-temp-slug-5486083`.
